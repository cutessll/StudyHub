# Final Demo Checklist

## Перед демонстрацією
1. Переконатися, що проєкт збирається: `dotnet build StudyHub.sln`.
2. Переконатися, що базові тести проходять: `dotnet run --project StudyHub.Tests/StudyHub.Tests.csproj`.
3. Запустити застосунок: `dotnet run --project StudyHub/StudyHub.csproj`.

## Сценарій демонстрації
1. Увійти як `guest` і показати: списки об'єктів та пошук.
2. Увійти як `student`:
   - додати матеріал;
   - знайти його через пошук;
   - додати/видалити з обраного;
   - видалити матеріал зі своїх.
3. Увійти як `moderator`:
   - створити нового користувача;
   - відредагувати об'єкт зі списку;
   - видалити матеріал;
   - заблокувати/видалити користувача.

## Що перевірити після демонстрації
1. Статус-рядок показує коректні повідомлення для кожної дії.
2. Лічильники користувачів/матеріалів оновлюються.
3. Після перезапуску застосунку збережений стан відновлюється.
